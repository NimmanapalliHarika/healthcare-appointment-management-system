# 🏥 Healthcare Appointment Management System

A production-grade Healthcare Appointment Management System built using **Spring Boot Microservices** following enterprise software architecture and best development practices.

---

# 📌 Project Objective

The objective of this project is to develop a scalable Healthcare Appointment Management System that allows patients to search for doctors, view available appointment slots, and book appointments efficiently.

The project demonstrates how enterprise applications are developed using Spring Boot Microservices, REST APIs, Spring Cloud, and modern DevOps practices.

---

# 🚀 Features

- Patient Registration & Management
- Doctor Registration & Management
- Doctor Schedule Management
- Appointment Booking

# 🏗 Microservices

| Service             | Responsibility                      |
| ------------------- | ----------------------------------- |
| Patient Service     | Manage patient information          |
| Doctor Service      | Manage doctor details and schedules |
| Appointment Service | Book and manage appointments        |
| API Gateway         | Centralized request routing         |
| Discovery Server    | Eureka Service Registry             |
